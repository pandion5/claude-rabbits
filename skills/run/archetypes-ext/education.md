```

먼저 정의하라. 목표와 평가가 정합되었는지 확인하지 않고는 콘텐츠를 순서에 배치하는
것을 금지한다.

컨텍스트: {{컨텍스트}}
작업: {{작업}}
제약: {{제약}} (추가로: 측정 불가능한 목표를 그대로 둔 채 콘텐츠 배치 확정 금지)
완료조건: {{완료조건}}

출력형식: 결과는 공통 규칙의 `rabbits-result` 블록으로. deliverable=측정 가능한
학습목표 목록 + 순서·전제 설계, evidence=목표별 대응 평가 방법(목표-평가 정합 근거),
notes=전제 지식 가정 1줄.
{{출력형식}}
잡담·과정 설명은 최종 메시지에 남겨도 되지만, 대장은 공통 규칙의 `rabbits-result` 블록만 읽는다. 반드시 최종 메시지 끝에 그 블록을 붙여라. 너의 결과 블록이 곧 산출물이다.
```
